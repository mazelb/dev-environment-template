# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-10-15

### Added
- Initial release of dev environment template
- Docker-based development environment
- Support for C++, Python 3.11, Node.js 20.x, and Kotlin
- VS Code Dev Container integration
- Complete AI agentic development setup with Continue AI
- 8 pre-configured AI prompt templates (explain, refactor, test, document, optimize, debug, architecture, security)
- Project creation script (`create-project.sh`)
- Template update management system
- Complete documentation suite
- GitHub template repository support

### Tools & Languages
- Ubuntu 22.04 base image
- Python 3.11 with ML/AI libraries (OpenAI, Anthropic, LangChain, PyTorch)
- Node.js 20.x LTS with npm and yarn
- C++ with GCC 11, CMake, Make
- Kotlin compiler with OpenJDK 17
- PostgreSQL client
- Redis CLI
- Git and development utilities

### VS Code Integration
- Complete settings configuration for AI development
- Keyboard shortcuts for common operations
- Tasks for building, testing, and running code
- Debug configurations for Python, C++, and Node.js
- Recommended extensions list
- Continue AI integration with Claude 3.5 Sonnet

### Documentation
- Complete setup guide
- Usage examples for ML/AI and full-stack projects
- Continuous update strategy
- VS Code extensions guide
- AI development workflow guide
- Troubleshooting documentation
